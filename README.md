# AI Assist Translator

AI Assist Translator is a simple speech-to-text translation web app built using Google AI tools.

Features:
- Speech input using microphone
- Automatic language detection
- Translation into English
- Translation history
- Clean and simple UI

Tech Stack:
- Google AI Studio
- Gemini AI
- Speech-to-Text
- JavaScript
- HTML & CSS

Demo:
This project was built as part of the GUVI Build & Deploy Apps with Google AI Studio program.
